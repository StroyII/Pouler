# How to contribute

To add your brick to the **Pouler Wall**, it’s very simple!

Just need to :

1. Have a <a href="https://github.com">Github</a> account
2. Fork this repository
3. Make a local clone :
- `git clone https://github.com/your_account/Pouler`
4. Create a new branch : 
- `git checkout -b pouler/<your-programm-name>`
5. Create your programm (Make sure to add a little README)
6. Commit your changes : 
- `git add .`
- `git commit -m "<commit-name>"`
7. Push your pouler 
- `git push origin <your-branch-name>`
8. Create a Pull Request (Describe your pouler)
9. Wait to be verified !

# 

### Guidelines

Keep your branch name descriptive:
pouler/<feature-name> or fix/<bug-name>

Test your code if possible before submitting.

Be kind and respectful — we’re all here to build something cool together !
